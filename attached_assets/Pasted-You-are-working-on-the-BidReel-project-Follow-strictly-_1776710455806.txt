## 🎯 TASK: Implement Fixed Price (Buy Now) System

We want to support two sale types:

1. auction (existing)
2. fixed (new)

---

## 🧩 STEP 1 — DATABASE

Add required columns to auctions table:

```sql
ALTER TABLE auctions
ADD COLUMN sale_type text DEFAULT 'auction' CHECK (sale_type IN ('auction', 'fixed')),
ADD COLUMN fixed_price numeric,
ADD COLUMN status text DEFAULT 'active' CHECK (status IN ('active', 'reserved', 'sold'));
```

Rules:

* fixed_price must be NULL for auctions
* fixed_price required if sale_type = 'fixed'

### 1. Create Auction

* saleType: "auction" | "fixed"
* fixedPrice (required if fixed)

### 3. BUY NOW ENDPOINT

### Logic:

1. Fetch auction

2. Ensure:

   * sale_type === "fixed"
   * status === "active"

SET status = 'sold'
WHERE id = $1 AND status = 'active'
RETURNING *
